# План разработки Doom-клона на Си

## Конфигурация проекта

| Параметр | Значение |
|---|---|
| Язык | C11 (clang) |
| Графика | SDL2 2.32.10 |
| Рендер | Raycasting (DDA) |
| Аудио | SDL_mixer |
| Загрузка изображений | stb_image.h (vendor/) |
| Внутренний буфер рендера | 640×400 → окно 1280×800 |
| Формат карты | ASCII-сетка |
| Управление | Doom-классика: W/S движение, A/D поворот, `,`/`.` стрейф, мышь обзор, ESC выход |
| FOV | 66° (plane 0.66, dir 1.0) |
| Скорость движения | 3.0 кл/с |
| Скорость поворота | 3.0 рад/с |
| Чувствительность мыши | 0.002 рад/px |
| Радиус игрока (коллизии) | 0.2 |
| Игровой цикл | фиксированный таймстепп 60 Гц для логики |
| Объём контента | Демо: 1 карта, 2-3 врага, 2 оружия |
| Уровень разработчика | Средний |

## Ключевые технические решения

1. **Игровой цикл с фиксированным шагом** — физика на 60 Гц, рендер каждый кадр. Избегает нестабильности на разных FPS.
2. **Z-buffer обязателен** — массив `float zBuffer[SCREEN_W]`, заполняется при raycasting стен, проверяется при рендере спрайтов.
3. **Коррекция fish-eye** — использовать `perpWallDist`, а не евклидово расстояние до стены.
4. **Коллизии** — проверять X и Y перемещения раздельно, чтобы скользить вдоль стены.
5. **ИИ врагов** — для демо достаточно преследования по прямой + рейкаст для проверки линии видимости (не A*, избыточно для 1 карты).
6. **Хитскан** — луч из камеры по направлению взгляда, поиск ближайшего спрайта, попадание по AABB в экранных координатах.

## Управление

| Клавиша | Действие |
|---|---|
| W / ↑ | Движение вперёд |
| S / ↓ | Движение назад |
| A / ← | Поворот влево |
| D / → | Поворот вправо |
| , | Стрейф влево |
| . | Стрейф вправо |
| Мышь ( deltaX ) | Обзор |
| Пробел / ЛКМ | Огонь (неделя 3) |
| 1 / 2 | Переключение оружия (неделя 3) |
| ESC | Выход |

## Формат карты (ASCII)

```
#  - стена (тип 1)
D  - дверь (тип 2)
P  - точка старта игрока
.  - пусто (проходимо)
E  - враг (зарезервировано, неделя 3)
M  - аптечка (зарезервировано, неделя 4)
A  - патроны (зарезервировано, неделя 4)
```

Карта по умолчанию 24×24, внешние границы — стены.

## Сборка и запуск

```bash
cd doom-clone
make          # сборка
make run      # сборка + запуск
make clean    # очистка
```

Зависимости (macOS/Homebrew):
```bash
brew install sdl2 sdl2_mixer
```
